# Image-Filtering
A Java Application that can apply 7 filters and remove noise from the image. And can also crop and flip the imaget

Filter List : 
1. Greyscale
2. Negative
3. Sepia
4. Reddish
5. Greenish
6. Blueish
7. Black and White

For removing noise Gaussian Algorithm is used, which smoothens (blurs) the image
